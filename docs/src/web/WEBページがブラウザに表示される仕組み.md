# WEB ページがブラウザに表示される仕組み

## terms

- DNS
- IP アドレス
- ドメイン

## 概要

1. DNS サーバーへリクエスト

Web ブラウザに URL を入力すると、URL からドメインを解析し、クライアントから DNS サーバーに問い合わせる。

1. 名前解決

DNS サーバーは問い合わせられたドメインに対応する IP アドレスを検索し、クライアントに該当の IP アドレスを返す。

1. Web サーバーへリクエスト

DNS サーバーから得られた IP アドレスを使用して Web サーバーに対して
コンテンツ(Web サイトのページファイルなど)を送信するように HTTP リクエストを送信する。

1. コンテンツの転送

ページファイルのリクエストを受けた Web サーバーは、クライアントに要求されたコンテンツを HTTP レスポンスとして返す。
